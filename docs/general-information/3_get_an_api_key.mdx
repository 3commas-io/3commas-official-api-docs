## How to Create an API Key

<Steps>
  <Step title="Sign in to 3Commas." description={
    <> <a href="https://app.3commas.io/en/auth/login"> Sign in </a> to 3Commas and select <strong>3Commas API</strong> from the right-side menu.
    </>
   }
  />
  <Step title="Add new API token" description={
    <> On the 3Commas API settings page, click the <strong> <span style={{ color: "#13a8a1" }}>New API access token</span> </strong> button.
    </>
  } 
  />
  <Step title="Select a method for generating the API Key." description={
    <>
    <p>If you choose <strong>Self-generated</strong>, you must provide an <strong>RSA Private Key</strong>.</p> 
    <p>To learn how to generate an RSA Private Key, see the article <a href="/general-information/how-to-create-an-rsa-key-pair"> How to create an RSA Key pair.</a> </p>
    </>
    }
  />
   <Step title="Complete the required form fields." description={ 
    <> 
    Specify the required parameters, including your RSA Private Key if applicable, and select the necessary permissions for the API Key.
    </>
   }
  />
  <Step title="Create your API Key."
  />
</Steps>

<Informer variant="danger">
    After creating an API Key, store the information securely.
</Informer>
