Adds a note to an existing SmartTrade.

<Separator />

<StickyLayout.Root>
<StickyLayout.Main>

## Path Parameter
<ApiParam name='id' type='integer' id="id" required>
   Unique 3Commas ID for this SmartTrade entity.
</ApiParam>

## Body Parameter
<ApiParam name='note' type='string' id="note" required>
   Short note for this SmarTrade.
   
   Minlength: `1`, Maxlength: `300`.
</ApiParam>
</StickyLayout.Main>
<StickyLayout.Sticky>

## Example Request

```json title="POST /v2/smart_trades/12345678/set_note"
{
  "note": "You can add an any note to an existing SmartTrade, limited to 300 characters."
}
```
</StickyLayout.Sticky>
</StickyLayout.Root>

<StickyLayout.Root>
<StickyLayout.Main>
    ## Response Parameter
    If successful, the response includes an updated copy of the [SmartTrade](/smart-trade/0_smart_trade_entity.mdx) entity.
</StickyLayout.Main>
