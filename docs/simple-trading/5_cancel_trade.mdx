Cancels a trade by its UUID.

<Informer variant="info">
  This endpoint can be used to cancel an open trade that is still in progress. Use the <code>cancelable</code> attribute in [Trade Entity](/docs/smart-trade/simple_test) to check if a trade is eligible for cancellation.
</Informer>

<Separator />

<StickyLayout.Root>
  <StickyLayout.Main>

## Path Parameters

<ApiParam name="uuid" type="string" id="uuid" required>
  The unique identifier of the trade to cancel. Example: <code>f89e3f34-1db1-47f2-81cc-9b4e5c50d7e1</code>
</ApiParam>

  </StickyLayout.Main>
  <StickyLayout.Sticky>

## Example Request

<NoContentEndpoint method="POST" endpoint="/ver1/trades/f89e3f34-1db1-47f2-81cc-9b4e5c50d7e1/cancel" />

  </StickyLayout.Sticky>
</StickyLayout.Root>

## Response Parameters

If successful, returns the updated trade object with its cancellation status. See [Trade Entity](/docs/smart-trade/simple_test) for full attributes.

  </StickyLayout.Main>
