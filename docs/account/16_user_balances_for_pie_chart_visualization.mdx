  url="/ver1/accounts/:account_id/pie_chart_data"
  method="POST"
  permissions={
    <>
      <Permission title="Permission" description="ACCOUNTS_READ" />
      <Permission title="Security" description="SIGNED" />
    </>
  }
/>

Returns information about currency balances of a specific account using its ID, for visualization in a pie chart.

<Informer variant="info">
By replacing **account_id** with **summary**, you can use the **/ver1/accounts/summary/pie_chart_data** endpoint to retrieve aggregated data for all accounts where **available_include_in_summary** is set to **true**.
</Informer>

## Response Parameters

<ApiParam name='account_id' type='integer' id="account_id">
ID of the exchange account entity.
</ApiParam>

<ApiParam name='code' type='string' id="code">
The cryptocurrency code symbol.
</ApiParam>

<ApiParam name='coinmarketcapid' type='string' id="coinmarketcapid">
Unique identifier of the cryptocurrency on CoinMarketCap.
</ApiParam>

<ApiParam name='name' type='string' id="name">
Full name of the cryptocurrency.
</ApiParam>

The exact percentage share of this cryptocurrency in the account's portfolio.

<CollapsibleApiParam name='amount' type='string' id="amount">
The amount of cryptocurrency held in this account.
</CollapsibleApiParam>

<CollapsibleApiParam name='btc_value' type='string' id="btc_value">
The equivalent value of this cryptocurrency in BTC.
</CollapsibleApiParam>

<CollapsibleApiParam name='usd_value' type='string' id="usd_value">
The equivalent value of this cryptocurrency in USD.
</CollapsibleApiParam>
