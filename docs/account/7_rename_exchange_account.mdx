Updates a name for the exchange account entity using its ID.

<Separator />

<StickyLayout.Root>
<StickyLayout.Main>

## Path Parameter

<ApiParam name='account_id' type='integer' id="account_id" required>
    Unique 3Commas ID of this exchange account entity.
</ApiParam>

## Query Parameter

<ApiParam name='name' type='string' id="name" required>
    New name for this exchange account.
</ApiParam>
</StickyLayout.Main>

<StickyLayout.Sticky>
## Example Request

<NoContentEndpoint method="POST" endpoint="/ver1/accounts/11223344/rename?name=new%20name" />

</StickyLayout.Sticky>
