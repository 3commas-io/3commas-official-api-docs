## Query Parameters

<ApiParam name='type' type='string' id="type" required>
    Name of the exchange in the 3Commas system.

    Use endpoint [Supported markets list](/market-data/0_supported_markets_list.mdx) and parameter `market_code` to get value for parameter.

</ApiParam>

<ApiParam name="name" type="string" id="name" required>
  Name for exchange account.
</ApiParam>

<ApiParam name="api_key" type="string" id="api_key" required>
  API-key for this exchange account.
</ApiParam>

<ApiParam name='secret' type='string' id="secret" required>
    Secret-key for this exchange account.
</ApiParam>
</StickyLayout.Main>

## Response Parameters

<ApiParam name="type" type="string" id="type">
  The exchange code available to connect.
</ApiParam>

<ApiParam name="connected" type="boolean" id="connected">
  Indicates whether the exchange is currently connected `true` or not `false`
</ApiParam>

<ApiParam name="available" type="boolean" id="available">
  Secret-key for this exchange account.
</ApiParam>

<ApiParam name='unified' type='boolean' id="unified">
  Shows the exchange supports an unified account. 
  Using for Bybit exchange `true` is if it supports.
</ApiParam>
</StickyLayout.Main>
