## Query Parameters

<ApiParam name='market_code' type='integer' id="market_code" required>
    Unique exchange code in the 3Commas system. 
    
    Use endpoint [Supported markets list](/docs/Market%20data/1.%20Supported%20markets%20list.mdx) to get value for parameter <code>market_code</code>.
</ApiParam>

</StickyLayout.Main>

## Example request

```json
GET /ver1/accounts/market_pairs?market_code=binance_us
```
