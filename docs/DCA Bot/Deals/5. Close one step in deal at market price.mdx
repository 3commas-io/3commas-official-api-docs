<Endpoint
  url="/ver1/deals/:deal_id/panic_sell_step"
  method="POST"
  permissions={
    <>
      <Permission title="Permission" description="BOTS_WRITE" />
      <Permission title="Security" description="SIGNED" />
    </>
  }
/>

Closes a specific Trade within the Deal by market using its ID.

<Informer variant="warning">
    After this action, the final status **Panic_sold** will be automatically set by 3Commas.
</Informer>

<StickyLayout.Root>
<StickyLayout.Main>

## Path Parameter
<ApiParam name='deal_id' type='integer' id="deal_id" required>
  Unique 3Commas ID Deal entity.
</ApiParam>


## Body Parameter

<ApiParam name='trade_id' type='integer' id="trade_id" required>
  Unique 3Commas ID Trade of the deal that should be sold. Use endpoint [Get Deal](DCA%20Bot/Deals/Get%20Deal.md) to get value for parameter 'trade_id'.
</ApiParam>
</StickyLayout.Main>
<StickyLayout.Sticky>

### Example request

```
POST
/ver1/deals/12345678/panic_sell_step?trade_id=1123324654
```
</StickyLayout.Sticky>
</StickyLayout.Root>

<StickyLayout.Root>
<StickyLayout.Main>

### Parameter response

If successful, the response includes an updated copy of the [Deals](./README.md) entity.

</StickyLayout.Main>
