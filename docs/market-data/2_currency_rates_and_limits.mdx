## Query Parameters

      <ApiParam name='market_code' type='integer' id="market_code" required>
         Unique exchange code in the 3Commas system. 
         
         Use endpoint [Supported markets list](/market-data/0_supported_markets_list.mdx) to get value for parameter `market_code`.
      </ApiParam>

      <ApiParam name='pair' type='string' id="pair" required>
         Trading pair in 3Commas format. 
         
         Use endpoints [All Market Pairs](/market-data/1_all_market_pairs.mdx) to get value for parameter `pair` in 3Commas format.
      </ApiParam>

      <ApiParam name='limit_type' type='string' id="limit_type">
         Defines the type of trading limits returned based on the selected trading tool `bot` or `smart_trade`:
          <br/>
         * If set to `bot`, the response includes trading limits specific to DCA, Signal and Grid Bots
         * If set to `smart_trade`, the response includes trading limits for SmartTrade
      </ApiParam>
   </StickyLayout.Main>
