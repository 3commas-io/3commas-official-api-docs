The method allows getting the list of pairs supported by the exchange in 3Commas format.

<StickyLayout.Root>
   <StickyLayout.Main>

## Query Parameter

<ApiParam name='market_code' type='integer' id="market_code" required>
    Unique exchange code in the 3Commas system.
    
    Use endpoint [Supported markets list](/market-data/0_supported_markets_list.mdx) to get value for parameter `market_code`.
</ApiParam>

</StickyLayout.Main>
   <StickyLayout.Sticky>
   
## Example Request

```json
GET /ver1/accounts/market_pairs?market_code=binance_us
```
