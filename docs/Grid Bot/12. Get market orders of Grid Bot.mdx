Returns a list of market orders for a Grid Bot using its ID

<StickyLayout.Root>
<StickyLayout.Main>

## Path Parameter
<ApiParam name='id' type='integer' id="id" required>
   Unique 3Commas ID for this Grid Bot entity.
</ApiParam>

## Query Parameters

<ApiParam name='limit' type='integer' id="limit">
   Quantity of the records you want to get in response.
   
   Minimum: <code>1</code>, Maximum: <code>1000</code>, Default: <code>100</code>
</ApiParam>

<ApiParam name='offset' type='integer' id="offset">
   Used to specify the starting point for a set of records to return in a paginated list.
   
   Default: <code>0</code>
</ApiParam>
</StickyLayout.Main>
