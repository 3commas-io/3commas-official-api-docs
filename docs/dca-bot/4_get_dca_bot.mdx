<Endpoint
  url="/ver1/bots/:bot_id/show"

      <Permission title="Security" description="SIGNED" />

/>

 Shows information about a specific DCA Bot using its ID.

<Separator />

<StickyLayout.Root>
<StickyLayout.Main>

## Path Parameter

<ApiParam name='bot_id' type='integer' id="bot_id" required>
  Unique 3Commas ID for this DCA Bot entity.
</ApiParam>

## Query Parameter

<ApiParam name='include_events' type='boolean' id="include_events" required>
  Indicates whether to include information about events related to the entity (`true`) or not (`false`).
</ApiParam>
</StickyLayout.Main>
<StickyLayout.Sticky>

## Example Request

```json
GET /ver1/bots/12345678/show?include_events=true
```
</StickyLayout.Sticky>
</StickyLayout.Root>

<StickyLayout.Main>

## Response Parameters

 If successful, the response includes a copy of [DCA Bot](/dca-bot/0_dca_bot_entity.mdx) entity.
</StickyLayout.Main>
