Returns available strategy list for DCA Bot. Use the query parameters to page through results

<StickyLayout.Root>
<StickyLayout.Main>
## Query Parameters

<ApiParam name='account_id' type='integer' id="account_id">
    Unique 3Commas ID of the exchange account entity.

    Use the [List of connected exchanges and wallets](/docs/account/5_list_of_connected_exchanges_and_wallets.mdx) endpoint to retrieve the value for the `account_id` parameter.

</ApiParam>

<ApiParam name="type" type="string" id="type">
  Specifies the strategy direction for the bot
  <CollapsibleNestedParams openName="Hide allowed values" collapsedName="Show allowed values">
      * `long` - a strategy focused on buying low and selling high;
      * `short` - A strategy focused on selling high and buying low.
     </CollapsibleNestedParams>
</ApiParam>

<ApiParam name="strategy" type="string" id="strategy">
  Specifies the type of strategy list
  <CollapsibleNestedParams openName="Hide allowed values" collapsedName="Show allowed values">
      * `simple` - a single strategy type;
      * `composite` - a combination of multiple strategy types.
     </CollapsibleNestedParams>
</ApiParam>

</StickyLayout.Main>
