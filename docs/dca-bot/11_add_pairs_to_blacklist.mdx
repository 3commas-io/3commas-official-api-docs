<Endpoint
  url="ver1/bots/update_pairs_black_list"
  method="POST"
  permissions={
    <>
      <Permission title="Permission" description="BOTS_WRITE" />
      <Permission title="Security" description="SIGNED" />
    </>

Adds specified pair(s) to the blacklist, preventing all DCA bots from opening deals with these pairs.

<StickyLayout.Root>
<StickyLayout.Main>

## Body Parameter

<ApiParam name='pairs' type='string' id="pairs" required>
  Trading pair(s) in 3Commas format.
  
  Use the endpoint [All Market Pairs](/market-data/1_all_market_pairs.mdx) to get the value for parameter pair in 3Commas format.
</ApiParam>
</StickyLayout.Main>

<StickyLayout.Sticky>
## Example Request
```json
POST /ver1/bots/update_pairs_black_list?pairs=1INCH_1INCH-USD-SWAP
```
</StickyLayout.Sticky>
</StickyLayout.Root>

<StickyLayout.Root>
<StickyLayout.Main>
## Response Parameter

<ApiParam name='pairs' type='array' id="pairs">
  The list of pairs that was added to the blacklist.
</ApiParam>
</StickyLayout.Main>
