<Endpoint
  url="/ver1/deals/:deal_id/convert_to_smart_trade"
  method="POST"
  permissions={
    <>
      <Permission title="Permission" description="SMART_TRADE_WRITE" />
      <Permission title="Security" description="SIGNED" />
    </>
  }
/>

Allows to convert the deal entity into SmarTrade entity.

<Informer variant="warning">
    After this action, 3Commas will automatically set the final status to **Cancelled** for this deal.
</Informer>

<Separator />

<StickyLayout.Root>
<StickyLayout.Main>

## Path Parameter
<ApiParam name='deal_id' type='integer' id="deal_id" required>
  Unique 3Commas ID Deal entity.
</ApiParam>
   

## Body Parameter

<ApiParam name='stop_bot' type='boolean' id="stop_bot" required>
  Indicates whether to disable the DCA Bot (`true`) or not (`false`).
</ApiParam>
</StickyLayout.Main>
<StickyLayout.Sticky>

## Example Request

```json
POST /ver1/deals/1234567890/convert_to_smart_trade?stop_bot=false
```
</StickyLayout.Sticky>
</StickyLayout.Root>

<StickyLayout.Root>
<StickyLayout.Main>

## Response Parameters

If successful, the response includes a copy of [SmartTrade](/smart-trade/0_smart_trade_entity.mdx) entity.

</StickyLayout.Main>
