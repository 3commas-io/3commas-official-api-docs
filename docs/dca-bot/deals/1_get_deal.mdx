<StickyLayout.Main>

## Path Parameter

<ApiParam name='deal_id' type='integer' id="deal_id" required>
  Unique 3Commas ID Deal entity.
</ApiParam>
</StickyLayout.Main>

```json
GET /ver1/deals/2295490936/show
```
