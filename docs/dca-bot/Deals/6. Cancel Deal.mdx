<Endpoint
  url="/ver1/deals/:deal_id/cancel"
  method="POST"
  permissions={
    <>
      <Permission title="Permission" description="BOTS_WRITE" />
      <Permission title="Security" description="SIGNED" />

Cancels a specific deal within the DCA Bot using its ID.

<Informer variant="warning">
After this action, 3Commas will automatically set the final status to **Cancelled** for this deal.
</Informer>


<StickyLayout.Root>
<StickyLayout.Main>

## Path Parameter

<ApiParam name='deal_id' type='integer' id="deal_id" required>
  Unique 3Commas ID Deal entity.
</ApiParam>

</StickyLayout.Main>
<StickyLayout.Sticky>

## Example Request

```json
POST  /ver1/deals/2345678901/cancel
```
</StickyLayout.Sticky>
</StickyLayout.Root>

<StickyLayout.Root>
<StickyLayout.Main>

## Response Parameters

If successful, the response includes an updated copy of the [Deals](./README.md) entity.

</StickyLayout.Main>
