## Get SmartTrade<br>
<br>

**Description:** Shows information about a specific SmartTrade using its ID<br>

**Permission:** SMART_TRADES_READ<br>
**Security:** SIGNED<br>
<br>

<blockquote>

<code><mark style="color:blue"><strong> GET </strong></mark></code>

<code>/v2/smart_trades/{id}</code>

</blockquote>

<br>

### Path Parameters<br>
<p>
   <strong>id</strong>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<code>integer</code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<mark style="color:orange">required</mark><br>
   Unique 3Commas ID for the SmartTrade entity to be reduced
</p>
<br>

### Parameter response<br>

If successful, the response includes a copy of [SmartTrade](./README.md) entity.

<br>

### Example request<br>

```json
GET /v2/smart_trades/12345678
```
<br>
<br>
