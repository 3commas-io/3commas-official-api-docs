<p className="p-method-discription">
  Adds funds in SmartTrade, increasing the volume of the trade and averaging the price at any moment
</p>

<div class="alert alert--warning" role="alert" style={{ margin: '30px 20px 20px 20px', padding: '15px' }}>
   The endpoint is available for SmartTrades with statuses <code>waiting_position</code> and <code>waiting_targets</code>, but not for SmartTrades with trailing enabled, or for the types <code>simple_buy</code> and <code>simple_sell</code>.
</div>

<h2> Path Parameter </h2>
<ApiParam name='id' type='integer' id="id" required>
   Unique 3Commas ID for this SmartTrade entity
</ApiParam>

<h2> Body Parameters </h2>
<ApiParam name='order_type' type='integer' id="order_type" required>
   Order type for add funds in this SmartTrade: <code> market </code>, <code> limit </code>
</ApiParam>
<CollapsibleApiParam name="units" type="object" id="units" required>
    Defines the amount of units to add in this SmartTrade
   <CollapsibleNestedParams>
    <ApiParam name='value' type='number' id='value' required>
         Specifies the number of units to buy
    </ApiParam>
    </CollapsibleNestedParams>
</CollapsibleApiParam>
<CollapsibleApiParam name="price" type="object" id="price" required>
   Provides pricing details for the limit order. This parameter is required if <code>order_type</code> is set to <code>limit</code>.
   <CollapsibleNestedParams>
    <ApiParam name='value' type='number' id='value' required>
      Price for limit order
    </ApiParam>
    </CollapsibleNestedParams>
</CollapsibleApiParam>

<h2> Parameter response </h2>
<p>
    If successful, the response includes an updated copy of the [SmartTrade](/docs/SmartTrade/1.%20SmartTrade%20entity.md) entity.
</p>
